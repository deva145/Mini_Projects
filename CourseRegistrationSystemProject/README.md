# Course Registration System
A full-stack Course Registration System built using Spring Boot for backend and HTML, JavaScript for frontend, with MySQL as the database.


## Features
- View available courses
- Register a student for a course
- View enrolled students
- REST API–based backend
- Frontend communicates with backend using fetch() API
- Data stored in MySQL database


## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- REST APIs
- MySQL

### Frontend
- HTML
- JavaScript


## Project Structure
CourseRegistrationSystemProject/
├── backend/
├── frontend/
└── README.md


## How to Run

### Backend
1. Configure MySQL in application.properties
2. Run the Spring Boot application
3. Server runs on http://localhost:8080

### Frontend
1. Open index.html in browser or Live Server
2. Use UI to register and view courses
